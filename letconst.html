wstęp do js

let i const, nazwy zmiennych, temple string

const

const name = 'string' // zmienna (variable) stała

console.log(name) // wyświetli się w konsoli

let

let age = 30 // zmienna (variable), której wartość można zmieniać

console.log(age) // wyświetli się w konsoli

let można stworzyć bez podania wartości, const musi już mieć wartość

let color // nie musi mieć wartości
const city = 'Sosnowiec' // bez wartości wyświetli błąd składni w konsoli

nazwy zmiennych

  • nazwy zmiennych zawsze po angielsku
  • JS jest językiem case sensitive, czyli rozróżnia wielkość liter
  • zmiennych nie zaczyna się cyframi myślikami i kropkami
  • nazwy zmiennych większe niż jeden wyraz trzeba pisać notacją wielbłądzią
