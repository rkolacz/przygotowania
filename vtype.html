metody i własności na stringu - wszystkie na MDN

length - własność

console.log(text.length) // własność, która wypisze w konsoli sumę znaków w stringu w zmiennej text

toUpperCase i toLowerCase - metoda

console.log(text.toUpperCase()) // metoda musi zostać wywołana przez ()

const newText = text.toLowerCase()

console.log(newText) // lepiej stworzyć nową zmienną i dodać do niej metodę niz wywoływać motody konsolą

charAt i slice - metoda + dodawanie ich do siebie

const username = 'rafał';

const newUsername = username.charAt(0).toUpperCase() + username.slice(1); // metody można do siebie dodawać i je ze sobą łączyć

console.log(newUsername); // w konsoli pojawi sie wynik Rafał

number

const num1 = 1; // number zapisuje się poprostu wpisując liczbę

const num2 = '1';

const addNums = num1 + num2 // doda number num1 do string num2 i stworzy z nich string 11

console.log(typeof addNums) // potwierdzenie

metody i własności na number - wszystkie na MDN

parseInt - metoda do zamiany string na number

console.log(parseInt(addNums);// zamieni string na number 11

toFixed - metoda do zaokrąglania liczb

const num3 = 1.234;

const newNum3 = num3.toFixed(1);

console.log(newNum3); // konsola wyświetli zaokrągląną liczbę do (x) po przecinku
